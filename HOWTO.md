# HOW TO

## How to use qpdf

qpdf is a command line tool for transforming and inspecting PDF files. It can be used to linearize, encrypt, decrypt, and manipulate PDF files in various ways.

- It can be installed with `brew install qpdf`
- Source code repository is here [github](https://github.com/qpdf/qpdf)
- Manual page ia here [read the docs](https://qpdf.readthedocs.io/en/stable/)
